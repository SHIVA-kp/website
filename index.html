Client & Server Communication

Client sender a request to the server. The server processes and responds with data.
Protocols: HTTP, HTTPS

The Journey of a Web Request

  1. User types a URL
  2. DNS resolves sent to domain
  3. Request sent to server
  4. Server processes and displays data
  5. Browesr receives and displays data

HTTP1.1 and HTPP2 behave differently in request handling.
